개인정보 처리방침

바로팀 (QuickTeams) | 시행일: 2026년 2월 21일

1. 수집하는 개인정보

바로팀은 체육 수업 팀 편성을 위해 다음 정보를 앱 내에서 처리합니다.

  • 학생 이름, 학년, 반, 번호
  • 성별
  • 달리기 기록 (초 단위)

2. 데이터 저장 방식

모든 데이터는 사용자의 기기(iPhone/iPad) 내부에만 저장됩니다.

  • 로컬 SQLite 데이터베이스를 사용합니다.
  • 외부 서버로 전송되는 데이터는 없습니다.
  • 인터넷 연결 없이 완전히 오프라인으로 동작합니다.
  • 클라우드 동기화, 분석 도구, 광고 SDK를 사용하지 않습니다.

3. 제3자 제공

바로팀은 어떠한 개인정보도 제3자에게 제공하지 않습니다. 앱 내 데이터는 기기 밖으로 나가지 않습니다.

4. 데이터 보관 및 삭제

  • 데이터는 사용자가 직접 삭제할 때까지 기기에 보관됩니다.
  • 앱 내 설정에서 개별 학생 삭제 또는 전체 데이터 초기화가 가능합니다.
  • 앱을 삭제하면 모든 데이터가 함께 삭제됩니다.

5. 아동 개인정보 보호

바로팀은 교사가 수업 목적으로 사용하는 도구입니다. 학생이 직접 앱을 사용하거나 개인정보를 입력하지 않으며, 교사만이 데이터를 관리합니다.

6. 개인정보 처리방침 변경

본 방침이 변경되는 경우, 앱 업데이트를 통해 안내합니다.
